## ✨ 全新特性
🎯 **v2.0 重要升级**：
- **终端交互界面**：告别手动编辑代码，所有操作通过命令行完成
- **改用Qwen2.5-3B**：响应速度提升 40%，平均回复时间 ≤5s (RTX4060 8GB显存)
- **智能日志系统**：新增日志模式，记录存储到 `log.txt` 文件
- **Conda 一键部署**：通过 environment.yaml 快速复现环境

## 🚀 极速部署

### 一键环境配置
```bash
# 从 environment.yaml 创建环境
conda env create -f environment.yaml
```

### 如何启动
```bash
# 启动模型服务 (自动检测GPU)
python main.py

```
**使用终端**：
```bash
>>> 是否启用日志模式(Y/N): y
>>> 请输入要回复的目标: 测试账号
```

## ❓ 常见问题

- 重新运行程序键入监听对象

**Q：首次启动模型加载慢？**  
- 这是正常现象，模型需完全加载到内存

**Q：支持多语言回复吗？**  
- 当前默认支持中英文
